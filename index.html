About

보안 문제를 풀고, 재현하고, 기록합니다.

웹 보안과 CTF를 중심으로 문제 해결 경험을 쌓고 있습니다. 풀이 과정, 취약점 원인, 검증 방법을 남겨 다음 작업에 다시 활용하는 방식을 좋아합니다.

이 페이지는 CTF, 경진대회, CVE 기록을 빠르게 업데이트할 수 있도록 GitHub Pages 기반의 한 페이지 포트폴리오로 구성했습니다.
